## KBotDocs

**KBotDocs**는 프로그래밍 및 카카오톡봇 학습을 위한 자습서와 레거시 API, API2 등의 명세서를 제공하는 카카오톡봇 문서입니다. Issues / Pull requests를 이용해 누구나 제보 또는 기여할 수 있습니다.
