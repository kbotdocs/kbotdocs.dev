<Section>
    ## API2

    API2는 최근에 공개된 내장 API로, 레거시 API보다 더 체계적인 구조로 되어 있습니다. 카카오톡봇의 지원은 API2를 중심으로 이루어질 예정입니다.
</Section>
<Section>
    ## 객체

    <Table>
        <Thead>
            <Tr>
                <Th colSpan="2">API2의 객체</Th>
            </Tr>
        </Thead>
        <Tbody>
            <Tr>
                <Td><span>[`App`](/api2/App)</span></Td>
                <Td>
                    봇 구동 앱에 관한 기능들을 담고 있는 객체입니다.
                </Td>
            </Tr>
            <Tr>
                <Td><span>[`AppData`](/api2/AppData)</span></Td>
                <Td>
                    앱의 데이터에 관한 기능들을 위한 객체입니다.
                </Td>
            </Tr>
            <Tr>
                <Td><span>[`Bot`](/api2/Bot)</span></Td>
                <Td>
                    카카오톡봇을 추상화한 객체입니다.
                </Td>
            </Tr>
            <Tr>
                <Td><span>[`BotManager`](/api2/BotManager)</span></Td>
                <Td>
                    `Bot` 객체를 관리/보조해주는 객체입니다.
                </Td>
            </Tr>
            <Tr>
                <Td><span>[`Broadcast`](/api2/Broadcast)</span></Td>
                <Td>
                    다른 스크립트 간의 데이터 교류를 지원하는 객체입니다.
                </Td>
            </Tr>
            <Tr>
                <Td><span>[`Database`](/api2/Database)</span></Td>
                <Td>
                    스크립트 로컬 폴더의 `/Database` 폴더 내 데이터 조작에 관한 객체입니다.
                </Td>
            </Tr>
            <Tr>
                <Td><span>[`Device`](/api2/Device)</span></Td>
                <Td>
                    구동 환경/기기에 관한 정보를 제공하는 메소드들을 모은 객체입니다.
                </Td>
            </Tr>
            <Tr>
                <Td><span>[`Event`](/api2/Event)</span></Td>
                <Td>
                    API2의 이벤트들에 대한 상수들을 제공하는 객체입니다.
                </Td>
            </Tr>
            <Tr>
                <Td><span>[`FileStream`](/api2/FileStream)</span></Td>
                <Td>
                    내부 저장소의 파일을 다루는 객체입니다.
                </Td>
            </Tr>
            <Tr>
                <Td><span>[`GlobalLog`](/api2/GlobalLog)</span></Td>
                <Td>
                    앱 내 글로벌 로그에 로그를 기록할 수 있게 하는 객체입니다.
                </Td>
            </Tr>
            <Tr>
                <Td><span>[`Http`](/api2/Http)</span></Td>
                <Td>
                    웹 상에서의 request(리퀘스트; 요청)에 관한 기능을 제공하는 객체입니다.
                </Td>
            </Tr>
            <Tr>
                <Td><span>[`Log`](/api2/Log)</span></Td>
                <Td>
                    앱 내 스크립트 로그에 로그를 기록할 수 있게 하는 객체입니다.
                </Td>
            </Tr>
            <Tr>
                <Td><span>[`Security`](/api2/Security)</span></Td>
                <Td>
                    각종 암호화/복호화에 관한 기능들을 제공하는 객체입니다.
                </Td>
            </Tr>
        </Tbody>
    </Table>
</Section>
<Section>
    ## 이벤트

    API2는 이벤트 리스너가 지정되어 있지 않으며, 여러 이벤트가 존재합니다.

    <Table>
        <Thead>
            <Tr>
                <Th colSpan="2">API2의 이벤트</Th>
            </Tr>
        </Thead>
        <Tbody>
            <Tr>
                <Td><span>[`activityBackPressed`](/api2/activityBackPressed)</span></Td>
                <Td>
                    스크립트 액티비티의 [`onBackPressed`](https://developer.android.com/reference/android/app/Activity#onBackPressed())가 호출되면 발생하는 이벤트입니다.
                </Td>
            </Tr>
            <Tr>
                <Td><span>[`activityCreate`](/api2/activityCreate)</span></Td>
                <Td>
                    스크립트 액티비티의 [`onCreate`](https://developer.android.com/reference/android/app/Activity#onCreate(android.os.Bundle))가 호출되면 발생하는 이벤트입니다.
                </Td>
            </Tr>
            <Tr>
                <Td><span>[`activityDestroy`](/api2/activityDestroy)</span></Td>
                <Td>
                    스크립트 액티비티의 [`onDestroy`](https://developer.android.com/reference/android/app/Activity#onDestroy())가 호출되면 발생하는 이벤트입니다.
                </Td>
            </Tr>
            <Tr>
                <Td><span>[`activityPause`](/api2/activityPause)</span></Td>
                <Td>
                    스크립트 액티비티의 [`onPause`](https://developer.android.com/reference/android/app/Activity#onPause())가 호출되면 발생하는 이벤트입니다.
                </Td>
            </Tr>
            <Tr>
                <Td><span>[`activityRestart`](/api2/activityRestart)</span></Td>
                <Td>
                    스크립트 액티비티의 [`onRestart`](https://developer.android.com/reference/android/app/Activity#onRestart())가 호출되면 발생하는 이벤트입니다.
                </Td>
            </Tr>
            <Tr>
                <Td><span>[`activityResume`](/api2/activityResume)</span></Td>
                <Td>
                    스크립트 액티비티의 [`onResume`](https://developer.android.com/reference/android/app/Activity#onResume())가 호출되면 발생하는 이벤트입니다.
                </Td>
            </Tr>
            <Tr>
                <Td><span>[`activityStart`](/api2/activityStart)</span></Td>
                <Td>
                    스크립트 액티비티의 [`onStart`](https://developer.android.com/reference/android/app/Activity#onStart())가 호출되면 발생하는 이벤트입니다.
                </Td>
            </Tr>
            <Tr>
                <Td><span>[`activityStop`](/api2/activityStop)</span></Td>
                <Td>
                    스크립트 액티비티의 [`onStop`](https://developer.android.com/reference/android/app/Activity#onStop())가 호출되면 발생하는 이벤트입니다.
                </Td>
            </Tr>
            <Tr>
                <Td><span>[`batteryLevelChanged`](/api2/batteryLevelChanged)</span></Td>
                <Td>
                    앱 구동 기기의 배터리 잔량이 변화했을 때 발생하는 이벤트입니다.
                </Td>
            </Tr>
            <Tr>
                <Td><span>[`command`](/api2/command)</span></Td>
                <Td>
                    [`Bot.setCommandPrefix()`](/api2/Bot/setCommandPrefix)에 의해 정해진 접두어로 시작하는 명령어를 수신할 때 발생하는 이벤트입니다.
                </Td>
            </Tr>
            <Tr>
                <Td><span>[`message`](/api2/message)</span></Td>
                <Td>
                    메시지를 수신할 때 발생하는 이벤트입니다.
                </Td>
            </Tr>
            <Tr>
                <Td><span>[`notificationPosted`](/api2/notificationPosted)</span></Td>
                <Td>
                    알림이 발생할 때 발생하는 이벤트입니다.
                </Td>
            </Tr>
            <Tr>
                <Td><span>[`notificationRemoved`](/api2/notificationRemoved)</span></Td>
                <Td>
                    알림이 사라질 때 발생하는 이벤트입니다.
                </Td>
            </Tr>
            <Tr>
                <Td><span>[`startCompile`](/api2/startCompile)</span></Td>
                <Td>
                    컴파일 요청을 감지했을 때 컴파일 시작 전에 발생하는 이벤트입니다.
                </Td>
            </Tr>
            <Tr>
                <Td><span>[`tick`](/api2/tick)</span></Td>
                <Td>
                    매 틱(1초)마다 발생하는 이벤트입니다.
                </Td>
            </Tr>
        </Tbody>
    </Table>
</Section>
