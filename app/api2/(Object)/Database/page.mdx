<Section>
    ## Database

    `Database` 객체는 스크립트 로컬 폴더의 `/Database` 폴더 내 데이터 조작에 관한 객체입니다.
</Section>
<Section>
    ## 메소드

    <Table>
        <Thead>
            <Tr>
                <Th colSpan="2">`Database` 객체의 메소드</Th>
            </Tr>
        </Thead>
        <Tbody>
            <Tr>
                <Td><span>`Boolean`</span></Td>
                <Td>
                    [`exists`](/api2/Database/exists)`(fileName: String)` \
                    `fileName` 이름의 파일이 존재하는지의 여부를 반환합니다.
                </Td>
            </Tr>
            <Tr>
                <Td><span>`Object`</span></Td>
                <Td>
                    [`readObject`](/api2/Database/readObject)`(fileName: String)` \
                    `fileName` 이름의 파일의 내용을 `Object` 형식으로 반환합니다. 파일 내용의 형식이 `JSON`일 경우에만 가능합니다.
                </Td>
            </Tr>
            <Tr>
                <Td><span>`String`</span></Td>
                <Td>
                    [`readString`](/api2/Database/readString)`(fileName: String)` \
                    `fileName` 이름의 파일의 내용을 문자열로 반환합니다.
                </Td>
            </Tr>
            <Tr>
                <Td><span>`void`</span></Td>
                <Td>
                    [`writeObject`](/api2/Database/writeObject)`(fileName: String, obj: Object)` \
                    객체를 `JSON String`으로 변환한 값을 `fileName` 이름의 파일에 덮어씁니다. 파일이 존재하지 않을 경우 `fileName` 이름의 파일을 생성합니다.
                </Td>
            </Tr>
            <Tr>
                <Td><span>`void`</span></Td>
                <Td>
                    [`writeString`](/api2/Database/writeString)`(fileName: String, str: String)` \
                    문자열을 `fileName` 이름의 파일에 덮어씁니다. 파일이 존재하지 않을 경우 `fileName` 이름의 파일을 생성합니다.
                </Td>
            </Tr>
        </Tbody>
    </Table>
</Section>
