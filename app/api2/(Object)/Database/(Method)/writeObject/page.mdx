<Section>
    ## Database.writeObject()

    객체를 `JSON String`으로 변환한 값을 `fileName` 이름의 파일에 덮어씁니다. 파일이 존재하지 않을 경우 `fileName` 이름의 파일을 생성합니다.
</Section>
<Section>
    ## 구문

    ~~~js
    Database.writeObject(fileName: String, obj: Object);
    ~~~

    <Section>
        ### 매개변수

        <Dl>
            <Dt>`fileName`</Dt>
            <Dd>
                내용을 덮어쓰거나 새로 생성할 파일의 이름
            </Dd>
            <Dt>`obj`</Dt>
            <Dd>
                `JSON String`으로 변환 후 파일 내용으로 저장할 객체
            </Dd>
        </Dl>
    </Section>
</Section>
