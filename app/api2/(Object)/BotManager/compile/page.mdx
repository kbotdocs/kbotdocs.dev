<Section>
    ## BotManager.compile()

    특정 `Bot` 객체에 대응하는 스크립트를 컴파일합니다. `throwOnError?`의 값이 `true`일 경우 컴파일 에러 시 에러를 throw합니다.
</Section>
<Section>
    ## 구문

    ~~~js
    BotManager.compile(botName: String, throwOnError?: Boolean = false);
    ~~~

    <Section>
        ### 매개변수

        <Dl>
            <Dt>`botName`</Dt>
            <Dd>
                컴파일할 `Bot`의 이름
            </Dd>
            <Dt>`throwOnError?`</Dt>
            <Dd>
                컴파일 에러 시 에러 throw 여부
            </Dd>
        </Dl>
    </Section>
    <Section>
        ### 반환값

        `Boolean` - `botName`의 `Bot` 객체에 대응하는 스크립트의 컴파일에 성공할 경우 `true`, 실패할 경우 `false`를 반환합니다.
    </Section>
</Section>
