<Section>
    ## FileStream

    `FileStream` 객체는 내부 저장소의 파일을 다루는 객체입니다.
</Section>
<Section>
    ## 메소드

    <Table>
        <Thead>
            <Tr>
                <Th colSpan="2">`FileStream` 객체의 메소드</Th>
            </Tr>
        </Thead>
        <Tbody>
            <Tr>
                <Td><span>`String`</span></Td>
                <Td>
                    [`append`](/api2/FileStream/append)`(path: String, data: String)` \
                    `path` 경로의 파일 내용에 새로운 내용을 이어 씁니다. 이어 쓴 후 해당 파일의 최종 내용을 반환합니다.
                </Td>
            </Tr>
            <Tr>
                <Td><span>`Boolean`</span></Td>
                <Td>
                    [`copyFile`](/api2/FileStream/copyFile)`(path1: String, path2: String)` \
                    `path1` 경로의 파일을 `path2` 경로에 복사합니다. 파일 복사의 성공 여부를 반환합니다.
                </Td>
            </Tr>
            <Tr>
                <Td><span>`Boolean`</span></Td>
                <Td>
                    [`create`](/api2/FileStream/create)`(path: String)` \
                    `path` 경로의 파일을 생성합니다. 파일 생성의 성공 여부를 반환합니다.
                </Td>
            </Tr>
            <Tr>
                <Td><span>`Boolean`</span></Td>
                <Td>
                    [`createDir`](/api2/FileStream/createDir)`(path: String)` \
                    `path` 경로에 폴더를 생성합니다. 폴더 생성의 성공 여부를 반환합니다.
                </Td>
            </Tr>
            <Tr>
                <Td><span>`String`</span></Td>
                <Td>
                    [`getSdcardPath`](/api2/FileStream/getSdcardPath)`()` \
                    내장 메모리의 최상위 경로를 반환합니다.
                </Td>
            </Tr>
            <Tr>
                <Td><span>`Boolean`</span></Td>
                <Td>
                    [`moveFile`](/api2/FileStream/moveFile)`(path1: String, path2: String)` \
                    `path1` 경로의 파일을 `path2` 경로로 이동합니다. 파일 이동의 성공 여부를 반환합니다.
                </Td>
            </Tr>
            <Tr>
                <Td><span>`String`</span></Td>
                <Td>
                    [`read`](/api2/FileStream/read)`(path: String)` \
                    `path` 경로의 파일 내용을 문자열로 반환합니다.
                </Td>
            </Tr>
            <Tr>
                <Td><span>`Object`</span></Td>
                <Td>
                    [`readJSON`](/api2/FileStream/readJSON)`(path: String)` \
                    `path` 경로의 파일 내용을 `Object` 형식으로 반환합니다. 파일 내용의 형식이 `JSON`일 경우에만 가능합니다.
                </Td>
            </Tr>
            <Tr>
                <Td><span>`Boolean`</span></Td>
                <Td>
                    [`remove`](/api2/FileStream/remove)`(path: String)` \
                    `path` 경로의 파일을 삭제합니다. 파일의 삭제 성공 여부를 반환합니다.
                </Td>
            </Tr>
            <Tr>
                <Td><span>`void`</span></Td>
                <Td>
                    [`save`](/api2/FileStream/save)`(path: String, data: String, append?: Boolean = false)` \
                    문자열을 `path` 경로의 파일에 덮어씁니다. 파일이 존재하지 않을 경우 `path` 경로의 파일을 생성합니다. `append?`가 `true`일 경우 `path` 경로의 파일 내용에 새로운 내용을 이어 씁니다.
                </Td>
            </Tr>
            <Tr>
                <Td><span>`void`</span></Td>
                <Td>
                    [`saveJSON`](/api2/FileStream/saveJSON)`(path: String, json: Object)` \
                    객체를 `JSON String`으로 변환한 값을 `path` 경로의 파일에 덮어씁니다. 파일이 존재하지 않을 경우 `path` 경로의 파일을 생성합니다.
                </Td>
            </Tr>
            <Tr>
                <Td><span>`String`</span></Td>
                <Td>
                    [`write`](/api2/FileStream/write)`(path: String, data: String)` \
                    문자열을 `path` 경로의 파일에 덮어씁니다. 파일이 존재하지 않을 경우 `path` 경로의 파일을 생성합니다. 덮어쓰거나 생성한 파일의 최종 내용을 반환합니다.
                    <Noti type={"info"}>
                        채팅 자동응답 봇에서는 반환값이 존재하지 않습니다. (`void`)
                    </Noti>
                </Td>
            </Tr>
            <Tr>
                <Td><span>`void`</span></Td>
                <Td>
                    [`writeJSON`](/api2/FileStream/writeJSON)`(path: String, json: Object)` \
                    객체를 `JSON String`으로 변환한 값을 `path` 경로의 파일에 덮어씁니다. 파일이 존재하지 않을 경우 `path` 경로의 파일을 생성합니다.
                </Td>
            </Tr>
        </Tbody>
    </Table>
</Section>
