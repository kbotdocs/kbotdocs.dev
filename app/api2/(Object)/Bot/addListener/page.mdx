<Section>
    ## Bot.addListener()

    특정 이벤트에 대한 이벤트 리스너를 리스너 리스트의 마지막에 추가합니다. 해당 메소드의 동작은 [`Bot.on()`](/api2/Bot/on)와 동일합니다.
</Section>
<Section>
    ## 구문

    ~~~js
    bot.addListener(eventName: String, listener: (arg1?: Any, arg2?: Any, ... argN?: Any) => Any);
    ~~~

    <Section>
        ### 매개변수

        <Dl>
            <Dt>`eventName`</Dt>
            <Dd>
                리스너를 추가할 이벤트명
            </Dd>
            <Dt>`listener`</Dt>
            <Dd>
                추가할 이벤트 리스너
                - `arg1?, arg2?, ... argN?`: `eventName` 이벤트에 의해 주어지는 인자에 대한 매개변수입니다. API2의 이벤트는 [여기](/api2#이벤트)를 참고하세요.
            </Dd>
        </Dl>
    </Section>
</Section>
