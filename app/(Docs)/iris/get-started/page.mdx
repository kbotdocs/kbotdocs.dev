<Section>
    ## 요구사항

    Iris를 사용하기 위해 아래의 사항이 요구됩니다.

    - 카카오톡이 설치되어 있는 Android 기기.
    - Android 버전 및 보안 설정에 따라 카카오톡 데이터베이스 및 일부 시스템 서비스에 접근하기 위한 Root 권한.
    - Iris와 상호작용하기 위한 HTTP 서버.
</Section>
<Section>
  ### 설치

  1. [Github release](https://github.com/dolidolih/Iris/releases)에서 최신 버전의 `Iris.apk`와 `iris_control`을 다운로드합니다. Windows를 사용할 경우, `iris_control` 대신 `iris_control.ps1`을 다운로드합니다.
  2. `adb`를 사용해 `Iris.apk`를 Android 환경(`/data/local/tmp`)에 복사합니다.
  ~~~bash
  adb push Iris.apk /data/local/tmp
  ~~~
  3. 운영체제에 따라 적절한 방법을 택해 APK 파일을 실행합니다.

  <Tab label={["Windows", "Mac/Linux"]}>
    <TabItem>
      아래 명령어를 사용해 Iris를 실행할 수 있습니다.
      ~~~bash
      ./iris_control.ps1 install
      ./iris_control.ps1 start
      ~~~
    </TabItem>
    <TabItem>
      `iris_control`을 실행가능한 상태로 만듭니다.
      ~~~bash
      chmod +x iris_control
      ~~~
      아래 명령어를 사용해 Iris를 실행할 수 있습니다.
      ~~~bash
      ./iris_control install
      ./iris_control start
      ~~~
    </TabItem>
  </Tab>
  `iris_control`은 이외에도 `status`, `stop`, `install_redroid`을 제공합니다.
</Section>
<Section>
  ### 구성

  `http://[ANDROID_IP]:3000/dashboard`에 접속해 설정을 진행합니다. 구성 옵션은 다음과 같습니다.

  <ParamList caption={"구성 옵션"}>
    <ParamItem param={"Bot Name"} type={"string"}>
      봇의 이름입니다. 봇을 식별하기 위해 사용됩니다.
    </ParamItem>
    <ParamItem param={"Web Server Endpoint"} type={"string"}>
      새로운 카카오톡 메시지를 수신할 웹 서버의 URL입니다. (예: `http://172.17.0.2:5000/db for IrisPy`)
    </ParamItem>
    <ParamItem param={"DB Polling Rate"} type={"int"} defaultVal={"`100`"} optional>
      데이터베이스 변경 사항을 확인하는 지연 시간(밀리초; ms)입니다. (예: `200`)
      <Noti type={"warning"}>
        값이 낮을수록 데이터베이스의 변경 사항을 더 빈번하게 확인해 CPU 사용량이 증가할 수 있습니다.
      </Noti>
    </ParamItem>
    <ParamItem param={"Message Send Rate"} type={"int"} defaultVal={"`50`"} optional>
      카카오톡 메시지 전송 간 최소 간격(밀리초; ms)입니다. (예: `100`)
    </ParamItem>
    <ParamItem param={"Bot Port"} type={"int"}>
      HTTP 서버가 수신 대기할 포트 번호입니다. (예: `3000`)
    </ParamItem>
  </ParamList>
</Section>
