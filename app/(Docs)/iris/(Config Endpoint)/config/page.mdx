<Section>
  ## 응답

  ~~~json
  {
    "bot_name": "[YOUR_BOT_NAME]",
    "bot_http_port": [PORT_FOR_HTTP_SERVER],
    "web_server_endpoint": "[YOUR_WEB_SERVER_URL_FOR_MESSAGE_FORWARDING]",
    "db_polling_rate": [DATABASE_POLLING_INTERVAL_IN_MILLISECONDS],
    "message_send_rate": [MESSAGE_SEND_INTERVAL_IN_MILLISECONDS],
    "bot_id": [YOUR_KAKAO_TALK_USER_ID]
  }
  ~~~

  <ParamList caption={"본문 매개변수"}>
    <ParamItem param={"bot_name"} type={"string"}>
      봇의 이름입니다. 봇을 식별하기 위해 사용됩니다.
    </ParamItem>
    <ParamItem param={"bot_http_port"} type={"int"}>
      HTTP 서버가 수신 대기할 포트 번호입니다.
    </ParamItem>
    <ParamItem param={"web_server_endpoint"} type={"string"}>
      새로운 카카오톡 메시지를 수신할 웹 서버의 URL입니다.
    </ParamItem>
    <ParamItem param={"db_polling_rate"} type={"int"}>
      데이터베이스 변경 사항을 확인하는 지연 시간(밀리초; ms)입니다.
    </ParamItem>
    <ParamItem param={"message_send_rate"} type={"int"}>
      카카오톡 메시지 전송 간 최소 간격(밀리초; ms)입니다.
    </ParamItem>
    <ParamItem param={"bot_id"} type={"int"}>
      봇의 카카오톡 유저 Id입니다.
    </ParamItem>
  </ParamList>
</Section>
