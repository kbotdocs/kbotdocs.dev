        - 현재 `require`와 같은 외부 모듈 로드는 지원하지 않습니다.

<Section>
    ## V8

    **V8**는 StarLight용 플러그인으로, Rhino JS에 V8 엔진을 이용할 수 있게 합니다.
    실행 속도가 Rhino 엔진에 비해 **약 29~117배** 더 빠르다는 이점이 있습니다.
</Section>
<Section>
    ## 사용방법

    <Noti type="warning">
        사용하는 기기에 맞는 파일을 다운로드 해야 합니다.
        - [`v8-arm64-release.slp`](https://drive.google.com/file/d/15tREAqHLNF6YlFwR2QJAZ4YuAX2Ph-8W/view?usp=drive_link): 대부분의 기기와 휴대폰
        - [`v8-armeabi-release.slp`](https://drive.google.com/file/d/1UJzkVf75SsGnr4IZ9KnqARHfk8ibHBRh/view?usp=drive_link): 일부 구식 안드로이드 기기
        - [`v8-x86_64-release.slp`](https://drive.google.com/file/d/1k4rdIFOY11EbLDGOgPHHgOQH0rnbBA8M/view?usp=drive_link): 대부분의 에뮬레이터/가상화 기기
    </Noti>

    1. 기기에 맞는 `.slp` 파일을 `sdcard/StarLight/plugins/` 폴더 내에 추가합니다.
    2. StarLight를 재시작합니다.
    3. 새 프로젝트 생성시 언어 선택 항목에서 **JavaScript(V8)** 항목을 선택합니다.

    <Noti type="info">
        `eval()` 함수의 사용 여부와 Java 클래스 수동 접근 여부는 프로젝트 설정에서 변경할 수 있습니다.
    </Noti>
</Section>
