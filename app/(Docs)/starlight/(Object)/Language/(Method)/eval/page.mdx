    # Language.eval()

    문자열로 표현된 코드를 컴파일 단계 없이 즉시 실행합니다. 코드 실행 결과를 반환합니다.
</Section>
<Section>
    ## 구문

    ~~~js
    language.eval(code);
    ~~~

    <Section>
        ### 매개변수

        <ParamList>
            <ParamItem param={"code"} type={"String"}>
                문자열로 표현된 코드
            </ParamItem>
        </ParamList>
    </Section>
    <Section>
        ### 반환값

        `Any?` - 코드 실행 결과를 반환합니다.
