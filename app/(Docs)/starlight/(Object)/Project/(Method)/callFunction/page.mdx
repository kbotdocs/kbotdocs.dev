    # Project.callFunction()

    ## 구문

    ~~~js
    Project.callFunction(name: String, args: Any[], onException: (throwable: java.lang.Throwable) => Any);
    ~~~

            </Dd>
            <Dt>`args`</Dt>
            <Dd>
                호출할 함수에게 전달할 인자
            </Dd>

            <Dd>
                예외가 발생할 때 호출할 콜백 함수
