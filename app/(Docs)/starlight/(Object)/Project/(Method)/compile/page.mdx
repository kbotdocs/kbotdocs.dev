<Section>
    # Project.compile()

    프로젝트를 컴파일하고 컴파일 성공 여부를 반환합니다. `throwException`의 값이 `true`일 경우 컴파일 에러 시 에러를 throw합니다.
</Section>
<Section>
    ## 구문

    ~~~js
    Project.compile();
    Project.compile(throwException);
    ~~~

    <Section>
        ### 매개변수

        <ParamList>
            <ParamItem param={"throwException"} type={"Boolean"} defaultVal={"`false`"} optional>
                컴파일 에러 시 에러 throw 여부
            </ParamItem>
        </ParamList>
    </Section>
    <Section>
        ### 반환값

        `Boolean` - 컴파일에 성공할 경우 `true`, 실패할 경우 `false`를 반환합니다.
    </Section>
</Section>
