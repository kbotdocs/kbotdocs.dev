<Section>
    ## Notification.state

    알림의 상태를 나타내는 상수입니다.
</Section>
<Section>
    ## 구문

    ~~~js
    Notification.state;
    ~~~

    <Section>
        ### 값

        `Number` - 알림의 상태를 나타내는 상수입니다. 관련 상수는 다음과 같습니다:
        - [`STATE_DISMISSED`](/Notification/STATE_DISMISSED)` = -1`: 알림이 제거된 상태일 경우
        - [`STATE_DEFAULT`](/Notification/STATE_DEFAULT)` = 0`: 알림이 아직 생성되지 않았을 경우
        - [`STATE_CREATED`](/Notification/STATE_CREATED)` = 1`: 알림이 생성되었을 경우
    </Section>
</Section>
