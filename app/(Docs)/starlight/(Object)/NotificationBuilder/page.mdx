<Section>
    # NotificationBuilder

    `NotificationBuilder` 객체는 알림의 생성을 위한 빌더(Builder) 객체입니다. 해당 객체는 직접 호출할 수 없습니다. [`Notification.create()`](/starlight/Notification/create)을 이용해 `NotificationBuilder` 객체를 반환받을 수 있습니다.
</Section>
