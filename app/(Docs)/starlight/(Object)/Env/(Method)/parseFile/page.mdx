<Section>
    # Env.parseFile()

    특정 Dotenv 파일에 대한 `java.util.Map<String, String>` 값을 반환합니다.
</Section>
<Section>
    ## 구문

    ~~~js
    Env.parseFile(path: String);
    ~~~

    <Section>
        ### 매개변수

        <Dl>
            <Dt>`path`</Dt>
            <Dd>
                `java.util.Map<String, String>` 형식으로 가져올 Dotenv 파일의 경로
            </Dd>
        </Dl>
    </Section>
    <Section>
        ### 반환값

        `java.util.Map<String, String>` - `path` 경로의 Dotenv 파일에 대한 `java.util.Map<String, String>` 값을 반환합니다.
    </Section>
</Section>
