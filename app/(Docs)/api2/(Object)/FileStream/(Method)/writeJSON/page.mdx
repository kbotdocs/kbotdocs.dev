<Section>
    ## FileStream.writeJSON()

    객체를 `JSON String`으로 변환한 값을 `path` 경로의 파일에 덮어씁니다. 파일이 존재하지 않을 경우 `path` 경로의 파일을 생성합니다. 해당 메소드의 동작은 [`FileStream.saveJSON()`](/api2/FileStream/saveJSON)과 동일합니다.
</Section>
<Section>
    ## 구문

    ~~~js
    FileStream.writeJSON(path: String, json: Object);
    ~~~

    <Section>
        ### 매개변수

        <Dl>
            <Dt>`path`</Dt>
            <Dd>
                내용을 덮어쓰거나 새로 생성할 파일의 경로
            </Dd>
            <Dt>`json`</Dt>
            <Dd>
                `JSON String`으로 변환 후 파일 내용으로 저장할 객체
            </Dd>
        </Dl>
    </Section>
</Section>
