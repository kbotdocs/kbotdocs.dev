<Section>
    # FileStream.save()

    문자열을 `path` 경로의 파일에 덮어씁니다. 파일이 존재하지 않을 경우 `path` 경로의 파일을 생성합니다. `append`가 `true`일 경우 `path` 경로의 파일 내용에 새로운 내용을 이어 씁니다.
</Section>
<Section>
    ## 구문

    ~~~js
    FileStream.save(path, data);
    FileStream.save(path, data, append);
    ~~~

    <Section>
        ### 매개변수

        <ParamList>
            <ParamItem param={"path"} type={"String"}>
                내용을 쓰거나 새로 생성할 파일의 경로
            </ParamItem>
            <ParamItem param={"data"} type={"String"}>
                생성할 파일의 내용
            </ParamItem>
            <ParamItem param={"append"} type={"Boolean"} defaultVal={"`false`"} optional>
                `path` 경로의 파일이 존재할 경우 이어쓰기 여부
            </ParamItem>
        </ParamList>
    </Section>
</Section>
