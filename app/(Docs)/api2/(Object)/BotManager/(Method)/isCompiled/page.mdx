<Section>
    # BotManager.isCompiled()

    `botName` 이름의 `Bot` 객체(스크립트)의 컴파일 완료 여부를 반환합니다.
</Section>
<Section>
    ## 구문

    ~~~js
    BotManager.isCompiled(botName: String);
    ~~~

    <Section>
        ### 매개변수

        <Dl>
            <Dt>`botName`</Dt>
            <Dd>
                컴파일 완료 여부를 확인할 `Bot` 객체의 이름
            </Dd>
        </Dl>
    </Section>
    <Section>
        ### 반환값

        `Boolean` - `botName` 이름의 `Bot` 객체(스크립트)의 컴파일이 완료된 경우 `true`, 완료되지 않은 경우 `false`를 반환합니다.
    </Section>
</Section>
