<Section>
    # App.runOnUiThread()

    UI쓰레드 상에서 특정 함수를 실행합니다.
</Section>
<Section>
    ## 구문

    ~~~js
    App.runOnUiThread(task: () => Any, onComplete: (error, result) => Any);
    ~~~

    <Section>
        ### 매개변수

        <Dl>
            <Dt>`task`</Dt>
            <Dd>
                UI쓰레드 상에서 실행할 함수
            </Dd>
            <Dt>`onComplete`</Dt>
            <Dd>
                결과를 처리할 함수
                - `error`: `task`함수 실행 도중 일어난 에러의 객체를 전달받는 인자
                - `result`: `task`함수의 반환값을 전달받는 인자
            </Dd>
        </Dl>
    </Section>
</Section>
