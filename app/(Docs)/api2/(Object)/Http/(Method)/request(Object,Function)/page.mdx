<Section>
    # Http.request(Object, Function)

    [`jsoup`](https://jsoup.org/)을 이용해 옵션에 따라 비동기 request를 보냅니다.
</Section>
<Section>
    ## 구문

    ~~~js
    Http.request(option, callBack);
    ~~~

    <Section>
        ### 매개변수

        <ParamList>
            <ParamItem param={"option"} type={"Object"}>
                보낼 요청의 옵션
                <Details summary={"`option` 정보"}>
                    ~~~json
                    {
                      "url": String,
                      "timeout": Number,
                      "method": String,
                      "headers": { /* Header */ }
                    }
                    ~~~
                    <ParamList>
                        <ParamItem param={"url"} type={"String"}>
                            request를 보낼 URL
                        </ParamItem>
                        <ParamItem param={"timeout"} type={"Number"} defaultVal={"`3000`"} optional>
                            request 타임아웃 (밀리초; ms)
                        </ParamItem>
                        <ParamItem param={"method"} type={"String"} defaultVal={"`\"GET\"`"} optional>
                            request 메소드
                        </ParamItem>
                        <ParamItem param={"headers"} type={"Object"} defaultVal={"`null`"} optional>
                            request 헤더
                        </ParamItem>
                    </ParamList>
                </Details>
            </ParamItem>
            <ParamItem param={"callback"} type={"(error, response, doc) => Any"}>
                request를 보낸 후 실행할 콜백 함수
                <Details summary={"매개변수"}>
                    <ParamList>
                        <ParamItem param={"error"} type={"java.lang.Exception?"}>
                            (에러 발생 시) 요청 중 발생한 에러를 나타내는 `java.lang.Exception`
                        </ParamItem>
                        <ParamItem param={"response"} type={"org.jsoup.Connection.Response?"}>
                            (요청 정상 처리 시) 요청에 대한 응답을 나타내는 [`org.jsoup.Connection.Response`](https://jsoup.org/apidocs/org/jsoup/Connection.Response.html)
                        </ParamItem>
                        <ParamItem param={"doc"} type={"org.jsoup.nodes.Document?"}>
                            (요청 정상 처리 시) 응답의 body에 대한 문서 형식의 값
                        </ParamItem>
                    </ParamList>
                </Details>
            </ParamItem>
        </ParamList>
    </Section>
