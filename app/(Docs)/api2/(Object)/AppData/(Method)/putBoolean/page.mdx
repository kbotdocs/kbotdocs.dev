<Section>
    ## AppData.putBoolean()

    앱 데이터에 `key: boolean` (`String: Boolean` 형식) 데이터를 씁니다.
</Section>
<Section>
    ## 구문

    ~~~js
    AppData.putBoolean(key: String, boolean: Boolean);
    ~~~

    <Section>
        ### 매개변수

        <Dl>
            <Dt>`key`</Dt>
            <Dd>
                쓸 앱 데이터의 키
            </Dd>
            <Dt>`boolean`</Dt>
            <Dd>
                `key`에 대한 `Boolean`형 데이터
            </Dd>
        </Dl>
