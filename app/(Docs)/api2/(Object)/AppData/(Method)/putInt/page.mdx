<Section>
    ## AppData.putInt()

    앱 데이터에 `key: integer` (`String: Number(정수)` 형식) 데이터를 씁니다.
</Section>
<Section>
    ## 구문

    ~~~js
    AppData.putInt(key: String, integer: Number);
    ~~~

    <Section>
        ### 매개변수

        <Dl>
            <Dt>`key`</Dt>
            <Dd>
                쓸 앱 데이터의 키
            </Dd>
            <Dt>`integer`</Dt>
            <Dd>
                `key`에 대한 `Number`(정수)형 데이터
            </Dd>
        </Dl>
    </Section>
</Section>
