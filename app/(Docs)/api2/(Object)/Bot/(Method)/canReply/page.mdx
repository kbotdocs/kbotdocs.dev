<Section>
    # Bot.canReply()

    특정 방에 메시지를 발신할 수 있는지의 여부를 반환합니다. 발신 가능 여부는 특정 방에 대한 알림 세션이 존재하는지의 여부로 구별합니다.
</Section>
<Section>
    ## 구문

    ~~~js
    bot.canReply(room);
    bot.canReply(room, packageName);
    ~~~

    <Section>
        ### 매개변수

        <ParamList>
            <ParamItem param={"room"} type={"String"}>
                삭제할 앱 데이터의 키
            </ParamItem>
            <ParamItem param={"packageName"} type={"String"} defaultVal={"`null`"} optional>
                메시지 발신 가능 여부를 확인할 방이 소속한 메신저 앱의 패키지명
            </ParamItem>
        </ParamList>
    </Section>
    <Section>
        ### 반환값

        `Boolean` - `room` 방에 메시지를 발신할 수 있을 경우 `true`, 발신할 수 없을 경우 `false`를 반환합니다.
    </Section>
</Section>
