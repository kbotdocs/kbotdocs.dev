<Section>
    # Bot.off()

    특정 이벤트에 대한 이벤트 리스너를 제거합니다. `listener` 인자가 없을 경우 리스너 이벤트의 마지막 리스너를 제거합니다. 해당 메소드의 동작은 [`Bot.removeListener`](/api2/Bot/removeListener)와 동일합니다.
</Section>
<Section>
    ## 구문

    ~~~js
    bot.off(eventName);
    bot.off(eventName, listener);
    ~~~

    <Section>
        ### 매개변수

        <ParamList>
            <ParamItem param={"eventName"} type={"String"}>
                리스너를 제거할 이벤트명
            </ParamItem>
            <ParamItem param={"listener"} type={"(arg1, arg2, ..., argN) => Any"} optional>
                제거할 이벤트 리스너
                <Details summary={"매개변수"}>
                    <ParamList>
                        <ParamItem param={"arg1, arg2, ..., argN"} type={"Any"}>
                            `eventName` 이벤트에 의해 주어지는 인자에 대한 매개변수입니다. API2의 이벤트는 [여기](/api2#이벤트)를 참고하세요.
                        </ParamItem>
                    </ParamList>
                </Details>
            </ParamItem>
        </ParamList>
    </Section>
</Section>
