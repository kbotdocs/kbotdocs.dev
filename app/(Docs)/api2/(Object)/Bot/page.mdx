    # Bot

    `Bot` 객체는 카카오톡봇을 추상화한 객체입니다. 해당 객체는 직접 호출할 수 없습니다. [`BotManager.getCurrentBot()`](/api2/BotManager/getCurrentBot)을 이용해 스크립트에 대응되는 `Bot` 객체를 사용할 수 있습니다.
</Section>
