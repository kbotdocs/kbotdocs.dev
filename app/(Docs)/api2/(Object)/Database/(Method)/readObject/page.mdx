<Section>
    # Database.readObject()

    `fileName` 이름의 파일의 내용을 `Object` 형식으로 반환합니다. 파일 내용의 형식이 `JSON`일 경우에만 가능합니다.
</Section>
<Section>
    ## 구문

    ~~~js
    Database.readObject(fileName: String);
    ~~~

    <Section>
        ### 매개변수

        <Dl>
            <Dt>`fileName`</Dt>
            <Dd>
                내용을 `Object` 형식으로 읽을 파일 이름
            </Dd>
        </Dl>
    </Section>
    <Section>
        ### 반환값

        `Object` - `fileName` 이름의 파일의 내용을 `Object` 형식으로 반환합니다.
    </Section>
</Section>
