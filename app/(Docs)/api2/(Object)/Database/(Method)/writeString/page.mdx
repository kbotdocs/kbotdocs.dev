<Section>
    # Database.writeString()

    문자열을 `fileName` 이름의 파일에 덮어씁니다. 파일이 존재하지 않을 경우 `fileName` 이름의 파일을 생성합니다.
</Section>
<Section>
    ## 구문

    ~~~js
    Database.writeString(fileName, str);
    ~~~

    <Section>
        ### 매개변수

        <ParamList>
            <ParamItem param={"fileName"} type={"String"}>
                내용을 덮어쓰거나 새로 생성할 파일의 이름
            </ParamItem>
            <ParamItem param={"str"} type={"String"}>
                파일 내용으로 저장할 문자열
            </ParamItem>
        </ParamList>
    </Section>
