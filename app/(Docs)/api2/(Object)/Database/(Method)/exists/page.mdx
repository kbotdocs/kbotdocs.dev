<Section>
    ## Database.exists()

    `fileName` 이름의 파일이 존재하는지의 여부를 반환합니다.
</Section>
<Section>
    ## 구문

    ~~~js
    Database.exists(fileName: String);
    ~~~

    <Section>
        ### 매개변수

        <Dl>
            <Dt>`fileName`</Dt>
            <Dd>
                존재 여부를 확인할 파일의 이름
            </Dd>
        </Dl>
    </Section>
    <Section>
        ### 반환값

        `Boolean` - `fileName` 이름의 파일이 존재할 경우 `true`, 존재하지 않을 경우 `false`를 반환합니다.
    </Section>
</Section>
