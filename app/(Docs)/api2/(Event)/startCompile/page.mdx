<Section>
    # startCompile

    `startCompile` 이벤트는 컴파일 요청을 감지했을 때 컴파일 시작 전에 발생하는 이벤트입니다.
</Section>
<Section>
    ## 구문

    [`Bot.addListener()`](/api2/Bot/addListener)를 이용해 이벤트에 대한 리스너를 추가할 수 있습니다. [`Event.START_COMPILE`](/api2/Event/START_COMPILE)를 이용해 이벤트 이름을 전달할 수도 있습니다.
    ~~~js
    bot.addListener("startCompile", () => {});
    // 또는
    bot.addListener(Event.START_COMPILE, () => {});
    ~~~
</Section>
