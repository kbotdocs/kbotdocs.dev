<Section>
    # 개발 준비

    카카오톡봇을 개발하고 작동하기 위한 앱을 설치하고 첫 프로젝트를 생성하는 방법을 소개합니다.
</Section>
<Section>
    ## 앱 설치

    이전에 소개했듯이, 카카오톡봇을 위한 앱은 다양합니다. 이 문서에선 **메신저봇R**을 기준으로 합니다.

    1. 최신 메신저봇R의 배포는 Github Release에서 이루어집니다. [여기](https://github.com/MessengerBotTeam/msgbot-old-release/releases)에서 `messengerbot-0.7.36a-20241005-release.zip`을 다운로드합니다.
    <Noti type={"danger"}>
        Play 스토어에 존재하는 메신저봇R은 오래된 버전이며 더이상 업데이트를 제공하지 않습니다.
    </Noti>
    2. 다운로드한 압축 폴더를 압축해제한 후, 폴더 내부에 존재하는 APK 파일을 눌러 메신저봇R을 설치하세요.
    3. 구동 기기의 Android 버전이 7.0 미만일 경우, [**Wear OS by Google**](https://play.google.com/store/apps/details?id=com.google.android.wearable.app&hl=ko)를 추가로 설치해야 합니다. 설치 후 무언가를 추가로 할 필요는 없습니다.
</Section>
<Section>
    ## 첫 프로젝트 생성

    카카오톡봇은 프로젝트를 단위로 구분하고 작동합니다. 카카오톡봇을 개발하기 위해 프로젝트를 생성해야 합니다.

    1. 메신저봇R을 실행합니다.
    2. 우측 하단의 **+** 버튼을 클릭합니다.
    3. 프로젝트의 이름을 작성하고 **확인** 버튼을 클릭합니다. 다른 선택항목을 선택할 필요는 없습니다.
</Section>
