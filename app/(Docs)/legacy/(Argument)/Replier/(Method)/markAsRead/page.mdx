<Section>
    # Replier.markAsRead()

    메시지가 발신된 방의 메시지를 읽음 처리합니다. `room?`이 존재할 경우 `room?` 이름의 방에 대해 동작을 수행합니다. 방에 대한 알림 읽기 세션을 찾을 수 있는지의 여부를 반환합니다.
</Section>
<Section>
    ## 구문

    ~~~js
    replier.markAsRead(room?: String, packageName?: String);
    ~~~

    <Section>
        ### 매개변수

        <Dl>
            <Dt>`room?`</Dt>
            <Dd>
                메시지를 읽음 처리할 방의 이름
            </Dd>
            <Dt>`packageName?`</Dt>
            <Dd>
                메시지를 읽음 처리할 메신저 앱의 패키지명
            </Dd>
        </Dl>
    </Section>
    <Section>
        ### 반환값

        `Boolean` - 방에 대한 알림 읽기 세션을 찾을 수 있을 경우 `true`, 찾을 수 없을 경우 `false`를 반환합니다.
    </Section>
</Section>
