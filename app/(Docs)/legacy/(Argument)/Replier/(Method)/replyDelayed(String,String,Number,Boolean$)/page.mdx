<Section>
    # Replier.replyDelayed(String, String, Number, Boolean?)

    일정 시간 후 특정 방에 메시지를 발신하고 메시지를 발신할 방에 대한 세션의 존재 여부를 반환합니다. `hideToast`가 `true`일 경우 방의 세션 존재 여부를 토스트 메시지로 출력합니다.
</Section>
<Section>
    ## 구문

    ~~~js
    replier.replyDelayed(room, msg, delay);
    replier.replyDelayed(room, msg, delay, hideToast);
    ~~~

    <Section>
        ### 매개변수

        <ParamList>
            <ParamItem param={"room"} type={"String"}>
                메시지를 발신할 방의 이름
            </ParamItem>
            <ParamItem param={"msg"} type={"String"}>
                발신할 메시지의 내용
            </ParamItem>
            <ParamItem param={"delay"} type={"String"}>
                지연할 시간(밀리초; ms)
            </ParamItem>
            <ParamItem param={"hideToast"} type={"Boolean"} defaultVal={"`false`"} optional>
                메시지를 발신할 방의 세션 존재 여부를 토스트 메시지로 출력할 것인지의 여부
            </ParamItem>
        </ParamList>
    </Section>
    <Section>
        ### 반환값

        `Boolean` - `room` 방에 대한 세션이 존재하면 `true`, 존재하지 않으면 `false`를 반환합니다.
    </Section>
</Section>
