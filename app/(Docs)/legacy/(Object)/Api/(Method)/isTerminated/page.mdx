<Section>
    # Api.isTerminated()

    [`Api.interruptThreads()`](/legacy/Api/interruptThreads) 실행 이후 스크립트의 모든 쓰레드의 `interrupt` 처리가 완료되었는지의 여부를 반환합니다.
</Section>
<Section>
    ## 구문

    ~~~js
    Api.isTerminated(scriptName?: String);
    ~~~

    <Section>
        ### 매개변수

        <Dl>
            <Dt>`scriptName?`</Dt>
            <Dd>
                모든 쓰레드의 `interrupt` 처리가 완료되었는지의 여부를 확인할 스크립트의 이름
            </Dd>
        </Dl>
    </Section>
    <Section>
        ### 반환값

        `Boolean` - `scriptName?` 인자가 존재할 경우 `scriptName?` 이름의 스크립트에 대해 모든 쓰레드의 `interrupt` 처리가 완료되었으면 `true`, 아니면 `false`를 반환합니다.
        `scriptName?` 인자가 존재하지 않을 경우 해당 메소드를 호출한 스크립트에 대해 모든 쓰레드의 `interrupt` 처리가 완료되었으면 `true`, 아니면 `false`를 반환합니다.
    </Section>
</Section>
