<Section>
    # Api.makeNoti()

    상단바 알림을 생성합니다. 동일 ID의 알림이 존재할 경우 해당 알림을 수정합니다. 알림의 생성/수정 성공 여부를 반환합니다.
</Section>
<Section>
    ## 구문

    ~~~js
    Api.makeNoti(title, content);
    Api.makeNoti(title, content, id);
    ~~~

    <Section>
        ### 매개변수

        <ParamList>
            <ParamItem param={"title"} type={"String"}>
                알림의 제목
            </ParamItem>
            <ParamItem param={"content"} type={"String"}>
                알림의 내용
            </ParamItem>
            <ParamItem param={"id"} type={"Number"} defaultVal={"`0`"} optional>
                알림의 ID
            </ParamItem>
        </ParamList>
    </Section>
    <Section>
        ### 반환값

        `Boolean` - 알림의 생성 또는 수정에 성공할 경우 `true`, 실패할 경우 `false`를 반환합니다.
    </Section>
</Section>
