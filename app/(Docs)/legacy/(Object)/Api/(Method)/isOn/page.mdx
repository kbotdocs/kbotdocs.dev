<Section>
    ## Api.isOn()

    특정 스크립트의 활성화 여부를 반환합니다.
</Section>
<Section>
    ## 구문

    ~~~js
    Api.isOn(scriptName: String);
    ~~~

    <Section>
        ### 매개변수

        <Dl>
            <Dt>`scriptName`</Dt>
            <Dd>
                활성화 여부를 확인할 스크립트의 이름
            </Dd>
        </Dl>
    </Section>
    <Section>
        ### 반환값

        `Boolean` - `scriptName` 이름의 스크립트가 활성화 상태일 경우 `true`, 비활성화 상태일 경우 `false`를 반환합니다.
    </Section>
</Section>
