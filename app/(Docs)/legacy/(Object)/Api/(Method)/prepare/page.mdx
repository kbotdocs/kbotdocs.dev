<Section>
    # Api.prepare()

    모든 스크립트가 한번도 컴파일된 적이 없을 경우 컴파일하고 스크립트의 컴파일 상태를 나타내는 정수를 반환합니다. `scriptName?`이 존재할 경우 `scriptName?` 이름의 스크립트에 대해 동작을 수행합니다. 컴파일 에러가 발생할 경우 에러를 throw합니다.
</Section>
<Section>
    ## 구문

    ~~~js
    Api.prepare(scriptName?: String);
    ~~~

    <Section>
        ### 매개변수

        <Dl>
            <Dt>`scriptName?`</Dt>
            <Dd>
                컴파일할 스크립트의 이름
            </Dd>
        </Dl>
    </Section>
    <Section>
        ### 반환값

        `Number` - 스크립트의 컴파일 상태를 나타내는 정수를 반환합니다. 스크립트가 존재하지 않을 경우 `0`, 컴파일에 성공할 경우 `1`, 컴파일된 적이 있을 경우 `2`를 반환합니다.
    </Section>
</Section>
