<Section>
    # Api.markAsRead()

    특정 방의 메시지를 읽음 처리합니다. 방에 대한 알림 읽기 세션을 찾을 수 있는지의 여부를 반환합니다.
</Section>
<Section>
    ## 구문

    ~~~js
    Api.markAsRead(room);
    Api.markAsRead(room, packageName);
    ~~~

    <Section>
        ### 매개변수

        <ParamList>
            <ParamItem param={"room"} type={"String"}>
                메시지를 읽음 처리할 방의 이름
            </ParamItem>
            <ParamItem param={"packageName"} type={"String"} optional>
                메시지를 읽음 처리할 메신저 앱의 패키지명
            </ParamItem>
        </ParamList>
    </Section>
    <Section>
        ### 반환값

        `Boolean` - `room` 이름의 방에 대한 알림 읽기 세션을 찾을 수 있을 경우 `true`, 찾을 수 없을 경우 `false`를 반환합니다.
    </Section>
</Section>
