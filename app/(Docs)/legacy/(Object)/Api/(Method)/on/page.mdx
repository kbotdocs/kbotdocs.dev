<Section>
    # Api.on()

    스크립트의 작동을 활성화하고 스크립트의 존재 여부를 반환합니다.
</Section>
<Section>
    ## 구문

    ~~~js
    Api.on(scriptName?: String);
    ~~~

    <Section>
        ### 매개변수

        <Dl>
            <Dt>`scriptName?`</Dt>
            <Dd>
                활성화할 스크립트의 이름
            </Dd>
        </Dl>
    </Section>
    <Section>
        ### 반환값

        `Boolean` - `scriptName?` 인자가 존재할 경우 `scriptName?` 이름의 스크립트가 존재하면 `true`, 존재하지 않으면 `false`를 반환합니다.
        `scriptName?` 인자가 존재하지 않을 경우 무조건 `true`를 반환합니다.
    </Section>
</Section>
