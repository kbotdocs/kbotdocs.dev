<Section>
    ## FileStream.append()

    `path` 경로의 파일 내용에 새로운 내용을 이어 씁니다. 이어 쓴 후 해당 파일의 최종 내용을 반환합니다.
</Section>
<Section>
    ## 구문

    ~~~js
    FileStream.append(path: String, data: String);
    ~~~

    <Section>
        ### 매개변수

        <Dl>
            <Dt>`path`</Dt>
            <Dd>
                내용을 이어 쓸 파일의 경로
            </Dd>
            <Dt>`data`</Dt>
            <Dd>
                파일에 이어 쓸 내용
            </Dd>
        </Dl>
    </Section>
    <Section>
        ### 반환값

        `String` - `path` 경로의 파일에 이어 쓴 후 해당 파일의 최종 내용을 반환합니다.
    </Section>
</Section>
