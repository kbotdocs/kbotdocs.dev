<Section>
    ## FileStream.remove()

    `path` 경로의 파일을 삭제합니다. 파일의 삭제 성공 여부를 반환합니다.
</Section>
<Section>
    ## 구문

    ~~~js
    FileStream.remove(path: String);
    ~~~

    <Section>
        ### 매개변수

        <Dl>
            <Dt>`path`</Dt>
            <Dd>
                삭제할 파일의 경로
            </Dd>
        </Dl>
    </Section>
    <Section>
        ### 반환값

        `Boolean` - 파일 삭제에 성공할 경우 `true`, 실패할 경우 `false`를 반환합니다. `java.io.File.delete()`의 반환값과 같습니다.
    </Section>
</Section>
