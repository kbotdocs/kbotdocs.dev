<Section>
    ## Device.getBatteryTemperature()

    앱 구동 기기의 배터리 온도(*`temp`*` * 10`, ˚C)를 반환합니다. 해당 메소드는 채팅 자동응답 봇에서 지원하지 않으며 [`Device.getBatteryTemp()`](/legacy/Device/getBatteryTemp)로 대체할 수 있습니다. `Device.getBatteryTemp()`의 반환값은 `(`*`temp`*` * 10)`이 아님을 유의하세요.
</Section>
<Section>
    ## 구문

    ~~~js
    Device.getBatteryTemperature();
    ~~~

    <Section>
        ### 반환값

        `Number` - 앱 구동 기기의 배터리 온도(*`temp`*` * 10`, ˚C)를 반환합니다.
    </Section>
