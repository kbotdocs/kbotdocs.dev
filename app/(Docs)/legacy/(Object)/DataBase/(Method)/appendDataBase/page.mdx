<Section>
    # DataBase.appendDataBase()

    `fileName` 이름의 파일 내용에 새로운 내용을 이어 씁니다. 이어 쓴 후 해당 파일의 최종 내용을 반환합니다.
</Section>
<Section>
    ## 구문

    ~~~js
    DataBase.appendDataBase(fileName, data);
    ~~~

    <Section>
        ### 매개변수

        <ParamList>
            <ParamItem param={"fileName"} type={"String"}>
                내용을 이어 쓸 파일의 이름
            </ParamItem>
            <ParamItem param={"data"} type={"String"}>
                파일에 이어 쓸 내용
            </ParamItem>
        </ParamList>
    </Section>
    <Section>
        ### 반환값

        `Boolean` - `fileName` 이름의 파일에 이어 쓴 후 해당 파일의 최종 내용을 반환합니다.
    </Section>
</Section>
