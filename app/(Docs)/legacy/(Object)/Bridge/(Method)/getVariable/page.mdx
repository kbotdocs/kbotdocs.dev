<Section>
    ## Bridge.getVariable()

    단순 자동응답에 대하여 특정 방에서 특정 이름을 가진 `[[변수]]`의 값을 반환합니다.
</Section>
<Section>
    ## 구문

    ~~~js
    Bridge.getVariable(room: String, name: String);
    ~~~

    <Section>
        ### 매개변수

        <Dl>
            <Dt>`room`</Dt>
            <Dd>
                `[[변수]]`의 값에 대응하는 방 이름
            </Dd>
            <Dt>`name`</Dt>
            <Dd>
                `[[변수]]`의 이름
            </Dd>
        </Dl>
    </Section>
    <Section>
        ### 반환값

        `String` - `room` 방에서 `name` 이름의 `[[변수]]`의 값을 반환합니다.
    </Section>
</Section>
