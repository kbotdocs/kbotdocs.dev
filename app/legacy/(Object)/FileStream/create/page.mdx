<Section>
    ## FileStream.create()

    `path` 경로의 파일을 생성합니다. 파일 생성의 성공 여부를 반환합니다.
</Section>
<Section>
    ## 구문

    ~~~js
    FileStream.create(path: String);
    ~~~

    <Section>
        ### 매개변수

        <Dl>
            <Dt>`path`</Dt>
            <Dd>
                생성할 파일의 경로
            </Dd>
        </Dl>
    </Section>
    <Section>
        ### 반환값

        `Boolean` - `path` 경로의 파일 생성에 성공할 경우 `true`, 실패할 경우 `false`를 반환합니다.
    </Section>
</Section>
