<Section>
    ## Api.compile(String, Boolean?)

    특정 스크립트를 컴파일하고 컴파일 성공 여부를 반환합니다. `throwOnError?`의 값이 `true`일 경우 컴파일 에러 시 에러를 throw합니다.
</Section>
<Section>
    ## 구문

    ~~~js
    Api.compile(scriptName: String, throwOnError?: Boolean = false);
    ~~~

    <Section>
        ### 매개변수

        <Dl>
            <Dt>`scriptName`</Dt>
            <Dd>
                컴파일할 스크립트의 이름
            </Dd>
            <Dt>`throwOnError?`</Dt>
            <Dd>
                컴파일 에러 시 에러 throw 여부
            </Dd>
        </Dl>
    </Section>
    <Section>
        ### 반환값

        `Boolean` - `scriptName` 이름의 스크립트의 컴파일에 성공할 경우 `true`, 실패할 경우 `false`를 반환합니다.
    </Section>
</Section>
