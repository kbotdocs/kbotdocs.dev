<Section>
    ## Api.interruptThreads()

    스크립트의 쓰레드에 `interrupt` 요청을 보냅니다. `scriptName?` 인자가 존재할 경우 `scriptName?` 이름의 스크립트의 쓰레드에 `interrupt` 요청을 보냅니다.
    `scriptName?` 인자가 존재하지 않을 경우 해당 메소드를 호출한 스크립트의 쓰레드에 `interrupt` 요청을 보냅니다.
</Section>
