<Section>
    ## Api.replyToID(String, String, Boolean)

    특정 ID로 식별되는 방에 메시지를 발신하고 방에 대한 세션의 존재 여부를 반환합니다. `hideToast`가 `true`일 경우 방의 세션 존재 여부를 토스트 메시지로 출력합니다.
</Section>
<Section>
    ## 구문

    ~~~js
    Api.replyToID(roomId: String, message: String, hideToast: Boolean = false);
    ~~~

    <Section>
        ### 매개변수

        <Dl>
            <Dt>`roomId`</Dt>
            <Dd>
                메시지를 발신할 방의 고유 ID
            </Dd>
            <Dt>`message`</Dt>
            <Dd>
                발신할 메시지의 내용
            </Dd>
            <Dt>`hideToast`</Dt>
            <Dd>
                메시지를 발신할 방의 세션 존재 여부를 토스트 메시지로 출력할 것인지의 여부
            </Dd>
        </Dl>
    </Section>
    <Section>
        ### 반환값

        `Boolean` - `roomId`로 식별되는 방에 대한 세션이 존재하면 `true`, 존재하지 않으면 `false`를 반환합니다.
    </Section>
</Section>
