<Section>
    ## Api.isCompiling()

    특정 스크립트의 컴파일이 진행 중인지의 여부를 반환합니다.
</Section>
<Section>
    ## 구문

    ~~~js
    Api.isCompiling(scriptName: String);
    ~~~

    <Section>
        ### 매개변수

        <Dl>
            <Dt>`scriptName`</Dt>
            <Dd>
                컴파일이 진행 중인지의 여부를 확인할 스크립트의 이름
            </Dd>
        </Dl>
    </Section>
    <Section>
        ### 반환값

        `Boolean` - `scriptName` 이름의 스크립트의 컴파일이 진행 중일 경우 `true`, 진행 중이지 않을 경우 `false`를 반환합니다.
    </Section>
</Section>
