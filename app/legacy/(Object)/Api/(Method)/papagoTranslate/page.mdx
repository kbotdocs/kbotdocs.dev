<Section>
    ## Api.papagoTranslate()

    입력된 텍스트에 대한 Papago NMT 번역 결과를 반환합니다. `errorToString?`의 값이 `true`일 경우 에러를 문자열로 반환합니다.
</Section>
<Section>
    ## 구문

    ~~~js
    Api.papagoTranslate(sourceLanguage: String, targetLanguage: String, content: String, errorToString?: Boolean = false);
    ~~~

    <Section>
        ### 매개변수

        <Dl>
            <Dt>`sourceLanguage`</Dt>
            <Dd>
                수행할 번역의 출발어
            </Dd>
            <Dt>`targetLanguage`</Dt>
            <Dd>
                수행할 번역의 도착어
            </Dd>
            <Dt>`content`</Dt>
            <Dd>
                번역을 수행할 텍스트
            </Dd>
            <Dt>`errorToString?`</Dt>
            <Dd>
                에러를 문자열로 반환할 것인지에 대한 여부
            </Dd>
        </Dl>
    </Section>
    <Section>
        ### 반환값

        `String` - `content`를 `sourceLanguage`에서 `targetLanguage`으로 번역한 결과를 반환합니다.
    </Section>
</Section>
