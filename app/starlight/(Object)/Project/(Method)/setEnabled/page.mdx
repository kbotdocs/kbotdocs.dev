<Section>
    ## Project.setEnabled()

    프로젝트의 활성화 상태를 설정하고 활성화 상태 변경의 성공 여부를 반환합니다. 성공 여부는 프로젝트의 컴파일 여부에 따라 달라집니다.
</Section>
<Section>
    ## 구문

    ~~~js
    Project.setEnabled(enabled: Boolean);
    ~~~

    <Section>
        ### 매개변수

        <Dl>
            <Dt>`enabled`</Dt>
            <Dd>
                프로젝트의 활성화 여부
            </Dd>
        </Dl>
    </Section>
    <Section>
        ### 반환값

        `Boolean` - 프로젝트의 활성화 상태 변경에 성공할 경우 `true`, 실패할 경우 `false`를 반환합니다.
    </Section>
</Section>
