<Section>
    ## 레거시 API
    
    레거시 API는 카카오톡봇 초창기부터 지원해오고 있는 내장 API입니다. 등장한지 오래되었으며, 더 이상 사용을 권장하지 않습니다. [최신 레벨의 API인 API2를 사용하세요](https://kbotdocs.dev/api2).
</Section>
